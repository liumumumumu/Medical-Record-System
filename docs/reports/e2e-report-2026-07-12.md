## 正式调用链

1. 前端调用 Spring Boot 注册或登录，获得 JWT。
2. 前端向 `POST /api/v1/cases` 提交 JSON 或 multipart 病例，接收 HTTP 202。
3. 前端轮询 `GET /api/v1/jobs/{jobId}`。
4. Spring Boot 以 remote 模式调用 Flask AI 服务。
5. 任务完成后前端读取 `GET /api/v1/cases/{caseId}`。
6. 病例与任务状态持久化到 MongoDB。
7. 历史、详情、人工编辑、报告和附件均使用正式 `/api/v1/**` 接口。
