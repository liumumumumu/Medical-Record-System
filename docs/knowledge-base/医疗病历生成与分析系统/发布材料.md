# 发布材料

## 一句话介绍

一个面向 PC 端课程演示的医疗病历录入、结构化生成与结果分析工作台。

## 项目说明

医疗病历生成与分析系统使用配置化表单收集患者基础信息、病情描述、检查与诊疗信息，通过清晰的录入、校验、生成和结果查看流程，展示医疗文本结构化系统的前端交互方案。

当前 v0.2 已完成前端演示闭环，真实后端分析、认证和文件处理仍在开发中。页面生成内容仅用于课程演示和信息整理，不构成医学诊断或治疗建议。

## 核心展示点

- PC 优先的 1700px 页面版心和 880px 病例任务区
- 配置化字段、默认 / 完整表单和分组管理
- 必填校验、年龄范围、错误聚焦和状态反馈
- 病例摘要、结构化病历、诊疗分析和检查资料结果页
- React Router 页面闭环与会话内结果恢复
- 自定义医疗工作台视觉，Ant Design 仅使用少量图标

## 截图 / 视频
